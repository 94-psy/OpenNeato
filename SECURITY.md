# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
This is a hobbyist robotics project involving physical hardware movement and Li-Ion batteries.
**DO NOT use this software if physical safety cannot be guaranteed.**

If you discover a security vulnerability (e.g., in the web interface or ROS bridge), please DO NOT open a public issue.
Instead, email the maintainer at: `felicecacca+openneato@gmail.com` (or your email).
